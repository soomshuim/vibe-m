## 4. Cross-Reference Rules

### 4.1 중복 서술 금지

- **원칙:** 규칙 본문은 SSOT 문서에만 존재
- **다른 문서:** 참조 링크 + 요약만 허용
- **예시:** CLAUDE.md의 체크리스트는 "(요약)" 표시 필수

### 4.2 충돌 해결

```
충돌 발생 시: SSOT 문서가 항상 우선
→ 다른 문서는 SSOT에 맞춰 수정
→ CHANGELOG에 "SSOT alignment" 기록
```

### 4.3 버전 동기화

- SSOT 문서 변경 시 → 참조 문서도 확인
- CLAUDE.md 체크리스트 → MASTER 변경 시 자동 드리프트 위험

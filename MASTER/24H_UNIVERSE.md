## 1. 24H Universe Bible

### 1.1 World Concept: Why "Time" as the Organizing Axis

> **"하루는 감정의 지도다."**

시간은 가장 보편적인 감정 트리거다. 누구나 새벽 4시의 고독, 오후 3시의 나른함, 밤 11시의 해방감을 안다. 24H Universe는 이 공유된 시간 감각을 음악으로 번역한다.

**Channel Promise:**
- 청자의 현재 시간에 맞는 음악을 제공
- "지금 이 순간"을 대변하는 한국어 가사
- 배경음악이 아닌, 의미 있는 청취 경험

---

### 1.2 Core Brand Rule: Same DNA, Controlled Variation

모든 Station은 동일한 DNA를 공유하되, 5개 슬롯(A/B/C/D/E) + Vocal Persona로 변주한다.

```
DNA (불변) + Slots (가변) = Station Identity
```

---

### 1.3 Constants vs Variables

| Category | Constants (불변) | Variables (가변) |
|----------|-----------------|------------------|
| **Language** | 한국어 가사 only | - |
| **Vocal Register** | Chest-dominant default | Falsetto/Husky/Airy/Powerful (concept-only toggle) |
| **Harmony Guard** | Single lead vocal, No stacked harmonies | - |
| **Chorus Rule** | Arrangement-only expansion | Density level varies |
| **Lyric Structure** | Metric/Ending Mirroring, Chorus Static | Vocabulary axis per track |
| **Exclude Base** | Airy, Falsetto, Harmonized, Backing vocals | +2-3 station-specific |
| **Genre** | Korean R&B umbrella | Sub-genre varies (Lo-fi/Alt/Acoustic/Soul) |
| **BPM** | - | 60-100 range by station |
| **Key/Mode** | 5 global buckets | Station picks from bucket |
| **Lead Instrument** | - | Slot A varies |
| **Rhythm Source** | - | Slot B varies |
| **Vocal Persona** | - | Male/Female by station |

### 2.3 Chorus Expansion Rules (Arrangement-Only)

모든 Station의 Chorus는 **arrangement density**로만 확장한다.

**Allowed (✅):**
- Drum pattern: half-time → full-time
- Bass: root only → walking/melodic
- Pad/Strings: thin → wide stereo swell
- Reverb: dry → wet bloom
- One held note (longer sustain than any verse note)
- Single-voice unison (end-line only, very low mix)

**Forbidden (❌):**
- Stacked vocal harmonies
- Choir-like backing
- Doubled lead vocal throughout
- EDM vocal processing
- Falsetto runs (unless concept-toggle)

### 3.2 Template B: Style Prompt Skeleton (≤800 chars)

```
Korean [Sub-genre] R&B, [BPM] BPM, [Key] [Mode]

[Slot A]-led, [Slot B], [Texture description]

[Vocal Persona] vocal. Raw, Solid, Direct, Dry, Unprocessed.
Chest voice dominant. No falsetto.

Verse: [Slot A] carries melody, minimal accompaniment, intimate.
Verse2: stronger dynamics encouraged, natural strain allowed.
Chorus: drums fill out, bass melodic, pad widens, 1 held note (longer sustain than any verse note).
Bridge: stripped to [element], builds to final chorus.
Outro: gradual fade, [Slot A] solo.

Lead vocal remains single and dominant throughout.
Chorus expansion by arrangement density and stereo width only.
If any unison: single-voice, end-line only, very low in mix.

Exclude: [Group 1], [Group 2], [Group 3]
```

**Character Count Target:** 600-900 chars (excluding Exclude line)

### 3.3 Template C: Lyrics Prompt Skeleton

```
[intro]
[instrumental]

[verse1]
[4행, 각 6-10음절]
[물성 오브젝트 1개 이상]
[마지막 행: 동사/형용사 종지]

[bridge1]
[Scene 2-3행 + Thesis 1행]
[Thesis: 현상 기반, 사람 의존 X]

[chorus]
[3-4행, 100% 고정]
[1-2행: 스냅샷 훅 (명사+동사)]
[3-4행: 정서 문장]

[verse2]
[V1과 음절/행/품사 종지 동일]
[마지막 2행: 감정 에스컬레이션]
[신체 반응 단어 포함]

[bridge2]
[Scene 변주 + Thesis 동일]

[chorus]

[outro]
[instrumental]
```

**Lyric Constraints:**
- Total: 100-120 words
- Per section: 4-6 lines
- Pure Korean only (no 영어/외래어)
- Metric Mirroring: V1=V2 음절 수
- Ending Mirroring: V1=V2 품사 종지
- Chorus Static: 100% identical repeat
- Bridge Anchor: Thesis fixed, Scene varies
- Vocabulary Independence: no overlap with other tracks

STATION QC CHECKLIST
═══════════════════════════════════════════════════════

─────────────────────────────────────────────────────
[ ] INTRO (0:00-0:20): No pronunciation slur, no mumble

| `MANAGER.md` | QC phases, Fail-fast rules |
| `ROLES.md` | AI role separation, validation enforcement |
| `CLAUDE.md` | Workflow checklists, auto-reference rules |

**Usage:**
1. Pick a Station from the Time Slot Table
2. Fill the Station Track Brief template
3. Generate Style Prompt using skeleton
4. Generate Lyrics using skeleton
5. Run Station QC Checklist
6. If all PASS → proceed to Suno generation
7. If any FAIL → regenerate and repeat QC

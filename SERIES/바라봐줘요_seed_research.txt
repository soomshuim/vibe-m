### B. 코드 진행 분석 (악보 기반)

특징: 확장 코드(Extended Chords) 과다 사용

Intro/Verse:
Dbadd9 → GbM7/Ab → Db/Ab → Bbm9 → Db/Eb → Eb7 → Ab9sus4

Pre-Chorus:
Dbadd9 → /B → Gbadd9 → B13 → Dbadd9 → /B → GbM9 → GbmM7/A

Chorus:
DbM9 → G7-5 → GbM9 → B13 → Db/Ab → Bb13-9 → Bb7-9-13 → Db/Eb → Ab9sus4

Cause → Effect:
- add9, M9, sus4 등 확장 코드 → 날카로운 해결감 없이 "떠다니는" 느낌 유발
- /B, /Ab 등 슬래시 코드 → 베이스 라인이 부드럽게 하행/상행 → 흐르는 듯한 전개
- Eb7 → Ab9sus4 반복 → 완전한 해결 회피 → 아련함 지속

---

### C. 멜로디 분석 (악보 기반)

| 특징 | 관찰 |
|------|------|
| 음역 | 좁음 (1옥타브 내외) |
| 진행 | Stepwise (순차 진행 위주), 도약 최소 |
| 리듬 | Speech-like, 말하듯 자연스러운 프레이징 |
| 꾸밈음 | 거의 없음 (기교 배제) |

Cause → Effect:
- 좁은 음역 + 순차 진행 → 무리 없이 따라 부를 수 있는 친숙함
- 기교 배제 → 가사 전달력 극대화, 담백함 강조
- Speech-like 리듬 → 편지/독백 느낌

---

### D. 보컬 스타일 분석

| 요소 | 관찰 |
|------|------|
| 발성 | Chest voice, 진성 위주 |
| 톤 | 담백, 건조, 호소력 |
| 기교 | 최소 (런/페이크 거의 없음) |
| 비교 | 크러쉬와 유사하나 더 절제됨 |

Cause → Effect:
- 기교 없는 보컬 → 가사가 직접 전달되는 느낌 → "진심"으로 인식
- Dry 톤 → 90년대 발라드 향수 자극
- 절제된 표현 → 감정 과잉 없이 울림 유발

---

### E. 프로덕션 분석

| 요소 | 관찰 |
|------|------|
| Lead | Electric Piano / Rhodes 계열 |
| Rhythm | 미니멀 드럼 (또는 드럼리스 구간 존재) |
| Bass | 부드러운 베이스, 존재감 낮음 |
| Guitar | 기타 솔로 (후반부) |
| 공간 | Reverb 적당, 과하지 않음 |

Cause → Effect:
- 건반 중심 → 따뜻하고 아날로그적 질감
- 미니멀 리듬 → 보컬에 집중 유도
- 기타 솔로 (후반) → 감정 고조 없이 여운으로 마무리

---

### F. 가사 구조 분석 (악보 기반)

핵심 훅:
"그간의 마음들이 모두 녹아내리면, 내가 꿈꿔왔던 것처럼 그렇게 나를 바라봐줘요"

영어 삽입:
"I'm the one who found your love"

Cause → Effect:
- 한국어 주체 + 영어 한 줄 삽입 → 감정 전환점 역할
- "녹아내리면" → 시간/거리의 물리적 해소 이미지
- "바라봐줘요" 반복 → 간절함의 직접 호소

---

### G. 왜 지루하지 않은가? (Repetition Tolerance)

1. 확장 코드의 미해결감 → 같은 진행도 "떠다니는" 느낌으로 지루함 방지
2. 보컬 다이나믹 미세 변화 → 벨팅 없이도 후반부 감정 상승
3. 기타 솔로 삽입 → 구조적 변화 없이 청각적 환기
4. 가사의 직접성 → 추상적 비유 없이 "바라봐줘요"로 귀결 → 명확한 감정 앵커

```
Korean Slow R&B Ballad, 67 BPM, Db Major

Male vocal, Raw, Dry, Unprocessed, Chest voice dominant
No falsetto, No runs, No melisma
Speech-like phrasing, Narrow melodic range
Precise articulation, clear consonants

Synth piano-led, supportive, warm pad texture
Minimal drums or drumless verses
Soft bass, low presence
Moderate reverb, room ambience
EQ balanced sound, clean mix

Verse2 subtle intensity increase by vocal weight only
Chorus: longer sustain on hook phrase, no belt
Bridge: single guitar line, no solo flourish

No harmony, no backing vocals, no doubles
Lead vocal remains single and dominant throughout
```
